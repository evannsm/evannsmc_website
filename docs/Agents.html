Use the abstract to improve the NAHS project entry:” This paper addresses the problem of safe control from signal temporal logic safety specifications for linear systems with bounded uncertainty in a static environment. We leverage the recently developed framework of interval signal temporal logic to accommodate uncertainty. We then propose a runtime assurance algorithm that, at each controller update step, considers the proposed input from a nominal controller and minimally adjusts the proposed input to maintain satisfaction of the safety specification at all times under all realizations of the uncertainty. Similar to related methods for control synthesis from temporal logic constraints with or without uncertainty, our algorithm requires solving a mixed integer linear program at each control update step. We equip our solution with necessary conditions to guarantee its feasibility for all time. The efficiencies afforded by interval signal temporal logic imply that our algorithm incurs a modest increase in the size of the program to accommodate uncertainty, and we demonstrate via experiments the real-time computational tractability of this approach. Moreover, we prove that, if a computation deadline is ever missed, our solution approach ensures that a safe backup input is always available. We demonstrate our approach on a miniature autonomous blimp.”

  4. Make RTD-RAX Render Like LaTeX \texttt{} Across Website

Goal

Ensure that every instance of RTD-RAX on the website is displayed in a LaTeX-style typewriter (monospace) format, similar to \texttt{RTD-RAX}.

Step 1 — Add CSS Class

Add the following CSS to the global stylesheet:

.tt {
  font-family: "Fira Code", "JetBrains Mono", "Courier New", monospace;
  font-size: 0.95em;
  letter-spacing: 0.03em;
}

Step 2 — Replace All Instances in HTML

Find every occurrence of:

RTD-RAX

Replace it with:

<span class="tt">RTD-RAX</span>

Step 3 — Scope of Replacement

Apply this change consistently across:

  • Home page
  • Project pages
  • Blog posts
  • Navigation / banners
  • Any dynamically rendered content (if applicable)

Step 4 — Important Notes

  • Do not use <code> tags unless specifically desired for semantic code blocks.
  • Always use the <span class="tt">...</span> approach for styling consistency.
  • Ensure no duplicate wrapping occurs (i.e., avoid nesting .tt inside .tt).

Step 5 — Optional (Advanced Consistency)

If using a component framework (React, etc.), create a reusable component:

const TT = ({ children }) => (
  <span className="tt">{children}</span>
);

Then replace instances with:

<TT>RTD-RAX</TT>

Result

All instances of RTD-RAX will visually match LaTeX \texttt{} styling across the site.
