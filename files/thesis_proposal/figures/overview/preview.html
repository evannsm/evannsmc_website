Proposal-overview SVG options

Option 1 · Triad Orbit (conceptual / title-slide)

Three thrusts as colored satellites orbiting the plant. Abstract, memorable, good for a cover slide.

Option 2 · Runtime-Assurance Stack (architectural / methods-overview slide)

Nested envelopes: performance inside safety inside security. Reads outside-in. Good for the "how it all fits" slide.

Option 3 · Hierarchical Signal Flow (rewired)

Mission / obstacles / disturbances → T2 planner → T1 LLC → plant; T3 is a tall supervisory block on the right that taps all signals and can override to a verified backup.

Option 4 · Continuous Autonomy Cycle (OODA-style, with gradients)

Observe → Plan (T2) → Control (T1) → Act (plant) → Observe … with T3 as the always-on central supervisor radiating monitor lines to every station. All five circles get matching radial gradients ("lit from within") for a polished look.

Option 5 · Continuous Autonomy Cycle (flat colors, no gradients)

Identical to Option 4 in layout, content, and arrows — the only difference is that every circle uses a flat color fill (no radial gradients anywhere). Better if you want a cleaner, more printable / slide-friendly look, or if Option 4's gradients feel too glossy.
